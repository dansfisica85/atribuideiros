Como usar o sistema de avaliação (Versão Local)

Selecione seu nome (Davi ou Giovanni) na tela inicial. Sua seleção será lembrada.
Selecione um candidato da lista suspensa.
Responda todas as questões clicando em um dos botões de pontuação. Seu progresso é salvo automaticamente no navegador.
Ao finalizar todas as questões, clique no botão "Finalizar Avaliação".
O relatório final combinará os dados das avaliações finalizadas *neste navegador*. Para um relatório combinado real, ambos os avaliadores precisam finalizar suas avaliações *usando esta mesma página/navegador*, ou os dados precisam ser combinados manualmente.
As avaliações finalizadas são registradas no histórico local.
Use o botão "Limpar Dados Locais" na tela de seleção para reiniciar o estado (incluindo histórico e progresso).
